# Template Authoring

Templates are React components that render from `ProfileDocument`. They should not hard-code sample content.

## Resume Templates

Add metadata to `resumeTemplateMetas` and a component to `resumeTemplateComponents` in `src/templates/resumeTemplates.tsx`.

Rules:

- Respect `settings.sectionLabels`.
- Respect `settings.hiddenSections`.
- Respect `settings.sectionOrder` when the layout is linear.
- Use `settings.themeColor` for accent color.
- Keep A4 sizing through the `resume-page` class.
- Avoid adding template ids that already exist.

## Portfolio Templates

Add metadata and a component in `src/templates/portfolioTemplates.tsx`.

Portfolio templates should highlight:

- selected projects
- tech stack
- links to repo/demo/video
- contact links
- concise summary

Static export is generated by `src/lib/portfolioExport.ts`. If a new template needs special static HTML, add a branch there using the same `templateId`.
